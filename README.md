

1. Load up sc2025_getdata.r file and download all the required data/mutations for the 2025 season
2. Load up pitcherDash.code.R file and upload all functions to the environment
3. Load up Pitcher Shiny.R to get the app up and running!

## Files

| File | Purpose |
|------|---------|
| `sc2025_getdata.R` | Data loading, mutation, shared objects |
| `pitcherDash.code.R` | Contains all the functions used to upload the visuals in the shiny app |
| `pitcherShiny.R` | Shiny app for the pitcher visuals |
